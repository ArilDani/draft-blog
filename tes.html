Bab 1: Mantra Bicara (Print)

Agar komputer kita hidup, kita harus membuatnya berbicara. Di Python, mantra untuk berbicara adalah print(). Ini seperti menggunakan megafon ajaib. Apapun yang kamu masukkan ke dalam tanda kurung dan tanda kutip, komputer akan mengucapkannya kembali padamu.

Editor Mantra #1
print("Halo Dunia! Aku penyihir koding!")

Bab 2: Kotak Harta Karun (Variabel)

Seorang penyihir hebat butuh tempat untuk menyimpan barang-barangnya. Bayangkan kamu memiliki banyak kotak kosong. Kamu bisa menempelkan label nama di kotak itu, lalu memasukkan sesuatu ke dalamnya.

Di Python, kotak ini disebut Variabel. Kamu bisa menyimpan teks, angka, atau bahkan monster (di dalam game!) di dalamnya.

Editor Mantra #2
# Kita siapkan kotaknya
nama_penyihir = "Budi si Hebat"
umur = 10

# Ayo kita lihat isi kotaknya
print("Nama saya:")
print(nama_penyihir)
print("Umur saya:")
print(umur)

Bab 3: Persimpangan Jalan (Logika If/Else)

Dalam petualangan, kamu sering bertemu jalan bercabang. "Jika cuaca cerah, aku pergi ke hutan. Jika tidak (Else), aku diam di kastil."

Komputer juga perlu membuat keputusan seperti itu. Kita menggunakan mantra if (jika) dan else (jika tidak) untuk mengajari komputer memilih jalan.

Editor Mantra #3
cuaca = "hujan"

if cuaca == "hujan":
    print("Buka payung ajaibmu!")
else:
    print("Pakai topi naga kerenmu!")

Bab 4: Buku Mantra Mini (Fungsi)

Kadang-kadang, ada mantra panjang yang harus kamu ucapkan berulang kali. Capek kan? Para penyihir pintar menulis mantra panjang itu di buku kecil dan memberinya judul pendek.

Di Python, ini disebut Fungsi (Function). Kamu membuat kumpulan perintah, memberinya nama, dan nanti kamu tinggal memanggil namanya saja!

Editor Mantra #4
# Membuat mantra baru bernama 'sapa_teman'
def sapa_teman(nama):
    print("Halo " + nama + "! Selamat datang di kastil.")

# Menggunakan mantranya berkali-kali dengan mudah!
sapa_teman("Rara")
sapa_teman("Dito")

Bab 5: Pabrik Robot Ajaib (Class & Object)

Ini adalah sihir tingkat lanjut! Bayangkan kamu ingin membuat pasukan robot kucing. Kamu tidak mungkin membuatnya satu per satu dari nol, kan?

Kamu butuh "Cetak Biru" atau desain gambarnya dulu. Di Python, desain ini disebut Class.

Setelah desain (Class) jadi, kamu bisa mencetak banyak robot nyata darinya. Robot nyata yang sudah jadi itu disebut Object. Keren kan?

Editor Mantra #5 (Tingkat Lanjut!)
# --- Desain Cetak Biru (Class) ---
class RobotKucing:
    # Ini resep untuk membuat robotnya
    def __init__(self, nama_robot, warna_robot):
        self.nama = nama_robot
        self.warna = warna_robot

    # Ini kemampuan robotnya
    def mengeong(self):
        print(self.nama + " si " + self.warna + " berkata: MEOONG!")

# --- Membuat Robot Nyata (Object) ---
# Kita buat robot bernama 'RoboOyen' berwarna 'Oranye'
robotku = RobotKucing("RoboOyen", "Oranye")

# Suruh robotnya mengeong!
robotku.mengeong()
