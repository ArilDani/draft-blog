Setiap mantra harus ditulis dengan hati-hati. Python itu sangat teliti. Dia sangat peduli dengan kerapian.

💡 Tips Penyihir: Python sensitif dengan spasi di awal baris (indentasi). Jika spasinya salah, mantranya bisa meledak (error)! Gunakan tombol TAB untuk membuat spasi ajaib.

Seorang penyihir butuh tempat untuk menyimpan ramuan dan barang-barangnya. Di Python, kita punya Kotak Ajaib yang disebut Variabel.

Kamu bisa memberi label nama pada kotaknya, lalu isi dengan apa saja!

Bab 3: Jenis Energi (Tipe Data)

Dunia sihir memiliki berbagai jenis energi. Ada energi angka, energi teks, dan energi kebenaran. Python perlu tahu energi apa yang sedang kamu pegang.

🔮 Jenis Energi:
1. Integer: Angka bulat (contoh: 10)
2. Float: Angka pecahan/koma (contoh: 145.5)
3. String: Teks atau mantra kata-kata (diapit tanda kutip)
4. Boolean: Benar atau Salah (True/False)

Bab 6: Sihir Pengambil Keputusan (If/Else)

Penyihir cerdas harus bisa mengambil keputusan. "Jika ada naga, lari. Jika tidak, jalan santai." Python menggunakan kata kunci if (jika) untuk berpikir.

Pernah dihukum guru menulis "Saya tidak akan nakal" 100 kali? Penyihir Python tidak perlu menulis manual. Kita gunakan sihir Loop untuk mengulang perintah secara otomatis!

Bab 9: Tas Ransel Ajaib (List)

Kalau Variabel adalah kotak kecil, List adalah Tas Ransel Ajaib. Tas ini bisa menampung BANYAK barang sekaligus secara berurutan.

Gulungan Mantra #9
Efek Mantra:

Bab 10: Peti Terkunci & Koleksi Unik (Tuple & Set)

Tuple itu seperti peti kaca yang dikunci. Isinya bisa dilihat, tapi tidak bisa diubah atau diganti.
Set itu seperti koleksi stiker. Tidak boleh ada stiker yang sama (kembar) di dalamnya.

Gulungan Mantra #10
Efek Mantra:

Bab 11: Buku Telepon Gaib (Dictionary)

Ingin mencari arti sebuah kata? Atau mencari pemilik tongkat sihir? Gunakan Dictionary. Kita menyimpan data berpasangan: Kunci (Key) dan Isinya (Value).

Gulungan Mantra #11
Efek Mantra:

Bab 12: Cetakan Golem (Class)

Penyihir tingkat lanjut bisa menciptakan makhluk hidup buatan! Class adalah cetakan atau resep utamanya. Dari satu cetakan ini, kita bisa membuat ratusan Golem (Object) yang berbeda namanya.

Gulungan Mantra #12
Efek Mantra:

Bab 13: Tameng Pelindung (Try/Except)

Terkadang mantra bisa gagal dan meledak. Untuk melindungimu dari ledakan (Error), gunakan tameng sihir bernama Try dan Except. Jika gagal, tameng akan menangkapnya!

Gulungan Mantra #13
Efek Mantra:

Bab 14: Memanggil Bantuan (Import)

Penyihir hebat tidak bekerja sendirian. Kita bisa memanggil bantuan dari penyihir lain yang sudah membuat alat canggih. Di Python, kita menggunakan mantra import.
